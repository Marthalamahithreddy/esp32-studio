# 🚀 Gen Alpha ESP32 Studio

A browser-based Arduino IDE with real-time circuit simulation and ESP32 integration.

## 🔧 Tech Stack
- Backend: Flask (Python)
- Frontend: HTML, CSS, JavaScript (Canvas/SVG)
- Hardware: ESP32, Arduino Framework
- Communication: REST APIs

## ⚙️ Features
- Compile and run Arduino/C++ code from browser
- ESP32 device interaction via backend APIs
- Real-time circuit simulation using Canvas/SVG
- Sketch management and versioning
- Serial monitoring support

## 🧠 Architecture
User (Browser) → Flask Backend → ESP32 Device  
                             → Circuit Simulator (Canvas)
